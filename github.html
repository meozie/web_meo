Github Việt Hóa

✨ **Github Việt Hóa** là công cụ giúp chuyển đổi giao diện Github sang tiếng Việt, giúp bạn dễ dàng quản lý code và làm việc hơn.

Phù hợp cho những bạn mới bắt đầu tiếp cận với Github mà chưa quen với các thuật ngữ tiếng Anh chuyên ngành.

  • 🇻🇳 Việt hóa toàn bộ các nút điều hướng chính
  • ⚡ Tương thích tốt với các phiên bản Chrome mới nhất
  • 🎨 Giữ nguyên bố cục gốc, không làm lỗi giao diện
  • 🌈 Cập nhật liên tục theo thay đổi của Github
Bước 1: Giải nén file ZIP
- Nhấn chuột phải vào file .zip
- Chọn Extract here hoặc Giải nén tại đây

Bước 2: Mở trang quản lý tiện ích
👉 Với Chrome:
- Mở trình duyệt, chọn biểu tượng ⋮ (3 chấm dọc) ở góc trên bên phải.
- Chọn Tiện ích mở rộng (Extensions).
- Hoặc gõ vào thanh địa chỉ: chrome://extensions/
- Bật Chế độ dành cho nhà phát triển (Developer mode) ở góc phải trên.

Bước 3: Tải tiện ích lên
- Nhấn nút Load unpacked (Tải tiện ích đã giải nén).
- Chọn thư mục đã giải nén ở Bước 1 và nhấn OK.

*Lưu ý: Nếu vào Github vẫn chưa thấy tiếng Việt, hãy nhấn nút F5.
